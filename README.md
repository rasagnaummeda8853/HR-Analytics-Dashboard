# HR Analytics Dashboard

## Project Overview

This project is an end-to-end HR Analytics solution developed using Python, SQL, Data Modeling, and Power BI. The objective is to analyze employee attrition, workforce demographics, overtime trends, and departmental performance to support data-driven HR decisions.

## Tools & Technologies

* Python (Pandas)
* SQL
* Power BI
* Star Schema Data Modeling

## Project Workflow

### 1. Data Cleaning (Python)

* Imported HR employee dataset
* Cleaned and transformed data
* Created Age Group categories
* Exported cleaned dataset for analysis

### 2. Data Analysis (SQL)

* Employee count analysis
* Department-wise analysis
* Attrition analysis
* Overtime analysis
* Workforce insights

### 3. Data Modeling (Power BI)

* Implemented Star Schema
* Created Fact and Dimension tables
* Built relationships between tables

### 4. Dashboard Development

Developed an interactive HR Analytics Dashboard featuring:

* Total Employees KPI
* Attrition Count KPI
* Attrition Rate KPI
* Average Age KPI
* Attrition by Department
* Gender Distribution
* Overtime Analysis
* Salary by Age Group
* Employee Distribution by Job Role
* Interactive Slicers

## Key Insights

* Total Employees: 1,470
* Attrition Rate: 16.12%
* Research & Development has the highest employee count.
* Employees working overtime show higher attrition.
* Attrition varies across departments and age groups.
